    Boolean

    Click the base piece (A), then the other (B). The result keeps A's name and settings; overlapping outlines must cross exactly twice. Subtract with B fully inside A punches it through as an internal cutout — a hole on the CUT layer.

    Stitch holes

    cm
    cm
    cm
    Diagonal slits along the edge. Click two edges to give both the same number of holes at matching positions; the count comes from the first edge ÷ spacing. Shift-click (or drag a box over) edges and guide lines to select many, then Enter or "Stitch selected" runs holes along each. For leather with no seam allowance, set Inset ≈ 0.3 so holes sit inside the cut edge.

    Cloud (GitHub)

    Patterns save as versioned files in your own GitHub repo. Create a repo, then a fine-grained token with Contents: read & write on just that repo. The token never leaves this browser.

      Quick help

      • Pen: click = corner, drag = curve, click first point to close · right-click = type exact length & angle
      • Select: drag points/handles · double-click point = corner↔smooth · double-click edge = add point
      • Click an edge, then type a length in the sidebar to resize it
      • Right-click an edge → divide it at a distance, percentage, or into equal parts
      • Weld: click an edge on each of two pieces to fuse them into one (the seam disappears)
      • Fold: select a straight edge → "Make fold line" — draft half, the mirror updates live
      • Inset: click an edge (or inside a piece) → guide line for stitching
      • Stitch: click two edges or guides → matched stitching slits on both
      • Knife: click two points (or an open path) to cut a piece in two
      • Boolean: overlap two pieces → union / subtract / intersect
      • Shape: drag rectangles/ellipses · dragging pieces magnets onto other points
      • Del removes point · scroll/two-finger pans · pinch or Ctrl+scroll zooms · Space+drag pans
      • Dashed line = cutting line (seam allowance)
